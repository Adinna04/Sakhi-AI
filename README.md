# 🌸 Sakhi AI

Sakhi AI is an AI-powered digital companion designed for women.

## Features

- 💬 AI Chat Assistant
- 🩸 Period Tracker
- 🧠 Mood Journal
- 🛡 Safety Hub
- 🎓 Scholarship Finder
- 💼 Career Coach
- ⚖ Legal Rights Guidance

## Tech Stack

- Python
- Streamlit
- Groq API
- SQLite

## Vision

To empower women through wellness, safety, education, and opportunities.
